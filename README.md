# Projecto Desarrollo Web React - VerveOS

Sistema POS (Point of Sale) para gastro-bares desarrollado con Next.js 16.

## Estructura

- **verve-os/** — Aplicación principal POS con panel de administración, gestión de pedidos, mesas, usuarios, cocina/barra, y dashboard de métricas.

## Tecnologías

- Next.js 16 con Turbopack
- TypeScript
- Tailwind CSS v4 + shadcn/ui
- Prisma + SQLite
- Socket.IO (WebSocket para pedidos en tiempo real)
- JWT (autenticación)
- Recharts (dashboard)
